# ZamStudio

ZamStudio is an AI-powered bilingual code editor concept built for Afghan developers, learners, classrooms, and small teams.

This public repository is intentionally product-facing. It contains safe public information only: brand overview, roadmap direction, and public project status.

## What ZamStudio Is

- A bilingual English and Pashto coding experience.
- A workspace concept for learning, building, and preparing projects.
- A planned Zeerak-powered assistant experience for developers.
- A future package-based product for individuals, teams, and classrooms.

## Public Status

ZamStudio is in active private development. Source code, deployment configuration, billing configuration, model integration details, and internal setup instructions are not published in this public repository.
