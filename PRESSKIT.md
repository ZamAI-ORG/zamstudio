## Short Description

ZamStudio is an AI-powered bilingual code editor concept for Afghan developers.

## Long Description

ZamStudio is being developed as a modern coding workspace with English/Pashto support, Zeerak assistant planning, package-based product direction, and learning-focused workflows for developers and classrooms.
