# ZamStudio Roadmap

Current Status: Planning & Initial Development

## Now (Q2 2026)
- [ ] **Core Editor Engine:** Implement Monaco-based code editor with basic syntax highlighting.
- [ ] **File System Integration:** Enable opening folders and managing project file trees.
- [ ] **AI Chat Panel:** Basic chat interface for querying the codebase (powered by Zeerak).
- [ ] **Theme System:** Initial light/dark mode support.

## Next (Q3 2026)
- [ ] **Intelligent Completion:** Context-aware code suggestions using local LLM models.
- [ ] **Plugin Architecture:** API for third-party extensions and themes.
- [ ] **Visual Designer:** Drag-and-drop UI builder for web components.
- [ ] **Git Integration:** Built-in commit, push, and diff viewer.

## Later (2027+)
- [ ] **Real-time Collaboration:** Google Docs-style multi-user editing.
- [ ] **Cloud Workspace Sync:** Seamless project state synchronization across devices.
- [ ] **Marketplace:** Public registry for plugins and templates.
- [ ] **Voice Coding:** Hands-free code dictation and navigation.
